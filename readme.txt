This is an extension for the CMS Contao (www.contao.org)
Tested in version 3.0.6

Purpose
You can create an "OnePage" in the Backend and add different sites from the page tree to it. Use the OnePage as a module or a content element. It will display all articles of the selected pages in the related Frontpages.

Installation
1. Copy the files to the directory system/modules/
2. Refresh your Database in the Contao Backend
